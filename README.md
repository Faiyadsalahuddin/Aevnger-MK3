# Aevnger-MK3
The Avenger MK3 Helmet Tracking Tank is an advanced remote controlled combat vehicle designed with real-time head-tracking technology for immersive control and targeting. The system combines a helmet mounted Gyro with wireless communication to control the movement and aiming system of the tank. The project uses an Arduino based control system, wireless transceivers, servo motors, and Gyro such as the MPU6050 to detect the operator’s head movements. When the user turns their head while wearing the helmet, the tank turret follows the same direction instantly, creating a first-person combat control experience which can change modern military targeting systems.
